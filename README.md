# 🛠️ **Treinamento Prático: Desenvolvendo uma Aplicação Fullstack com Fastify, Prisma, ViteJS, e SupaBase**

---

## 📋 **Sobre o Treinamento**

Bem-vindo(a) ao repositório oficial do nosso treinamento prático de desenvolvimento fullstack! 🎉

Durante esse treinamento, você aprenderá a construir do zero uma aplicação moderna e escalável usando tecnologias que estão em alta no mercado. Nosso foco é guiá-lo(a) desde a criação do backend com **Fastify** e **Prisma** até o desenvolvimento do frontend com **ViteJS** e **React**, culminando no deploy em produção com **Render** (backend) e **Vercel** (frontend). Também utilizaremos **PostgreSQL** localmente durante o desenvolvimento e a incrível plataforma **SupaBase** para banco de dados em produção.

Este repositório serve como guia e repositório central para o código desenvolvido durante o treinamento. Cada participante será capaz de explorar e entender como conectar todas as partes da aplicação e publicá-la em produção, utilizando ferramentas gratuitas e de fácil uso.

---

## 🚀 **Stacks e Ferramentas Utilizadas**

### 🔧 **Backend**
- **Fastify**: Um framework web focado em performance e flexibilidade. Sua arquitetura baseada em plugins facilita a manutenção e a escalabilidade de aplicações modernas.
- **Prisma ORM**: Um ORM que simplifica a interação com o banco de dados, trazendo autocompletar e tipagem estática no TypeScript. Isso ajuda a prevenir erros e melhorar a produtividade.
- **PostgreSQL**: O banco de dados relacional amplamente utilizado, com suporte local para desenvolvimento.
- **SupaBase**: Uma plataforma completa de backend-as-a-service, que oferece banco de dados PostgreSQL e APIs real-time de forma prática e escalável. Usaremos o plano gratuito para deploy em produção.
- **Render**: Plataforma de deploy simples e poderosa, onde faremos o deploy do backend.

### 🎨 **Frontend**
- **ViteJS**: Um build tool moderno, incrivelmente rápido, que facilita o desenvolvimento de aplicações React com uma configuração mínima.
- **React**: A biblioteca de UI mais popular para criar interfaces dinâmicas e interativas, baseada em componentes reutilizáveis.
- **Axios**: Biblioteca para fazer requisições HTTP no frontend, que utilizaremos para integrar com o backend.
- **Vercel**: Plataforma de deploy focada em frontend e serverless, perfeita para hospedar nosso frontend React com uma experiência de deploy sem complicações.

---

## 📚 **Conteúdo do Treinamento**

Durante as duas semanas de treinamento, vamos seguir um roteiro bem organizado que cobre tanto backend quanto frontend:

1. **Criação do Backend com Fastify e Prisma**
   - Estruturação do projeto backend com Fastify.
   - Modelagem de dados usando Prisma e PostgreSQL.
   - Implementação de rotas e CRUD no backend.

2. **Deploy do Backend**
   - Deploy do backend no Render.
   - Configuração do banco de dados em produção usando SupaBase.

3. **Criação do Frontend com ViteJS e React**
   - Estruturação do projeto frontend com ViteJS.
   - Integração do frontend com o backend Fastify usando Axios.
   - Criação de uma interface dinâmica e responsiva.

4. **Deploy do Frontend**
   - Deploy do frontend no Vercel.
   - Integração do frontend em produção com o backend no Render.

## 🌐 **Por que Escolhemos Essa Stack?**

A combinação de **Fastify**, **Prisma**, **PostgreSQL**, **ViteJS** e **React** não foi escolhida à toa. Essas ferramentas juntas proporcionam uma base sólida para o desenvolvimento de aplicações modernas, com foco em performance, simplicidade, e facilidade de manutenção.

- **Fastify**: Leve, rápido e com suporte a plugins, permitindo que a aplicação cresça sem comprometer a performance.
- **Prisma**: A integração simples com PostgreSQL e o suporte ao TypeScript tornam o processo de desenvolvimento mais seguro e eficiente.
- **ViteJS**: Por ser extremamente rápido e fácil de configurar, ViteJS é a escolha perfeita para desenvolver frontend de forma ágil.
- **SupaBase**: É uma alternativa open-source ao Firebase, com todas as funcionalidades necessárias para escalar a aplicação sem complexidade.
- **Render e Vercel**: Plataformas simples e gratuitas para deploy, permitindo que você coloque sua aplicação no ar sem complicações.
